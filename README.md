# Placement Passport — BN Clinical Portfolio

Single-file, 100% offline clinical placement logbook. Open `index.html` in any
browser (works great saved to a phone home screen, or on GitHub Pages). All
data stays in `localStorage` on the device.

## What it tracks

- **Hours** — placements (ward, practice area, dates, preceptor) and shifts
  (start/end/break, overnight-safe maths), with a progress bar against a
  configurable degree target (default 1100 h — the NZ BN minimum).
- **Skills passport** — a seeded, fully editable BN skills framework across
  Assessment / Personal cares / Medication / Infection control /
  Documentation / Communication / Emergency. Tap to cycle
  *not yet → practiced → signed off* (sign-off date recorded automatically).
- **Reflections** — structured Gibbs reflective cycle forms (all six stages
  with prompts), attachable to a placement.

## Export (⇩ button)

- **Print / PDF report** — a clean, signable portfolio document: summary,
  per-placement shift tables with hours, full skills matrix with sign-off
  dates, and complete reflections. Use the browser's print → Save as PDF.
- **JSON backup / restore** — download everything as one file; restore on a
  new device. **Back up before changing phones** — localStorage doesn't
  follow you.

Log shifts the same day and write one honest reflection a week: three years
of this is exactly the evidence base new-grad (NZNO portfolio) and eventual
NP applications are built from.
